# VSCode-Vim Cheat Sheet (Single Table)

| Category | Action | Key |
|--------|--------|-----|
| Mode | Insert mode | `i` |
| Mode | Normal mode | `Esc` |
| Mode | Visual (select text) | `v` |
| Mode | Visual line (select lines) | `V` |
| Move | Left | `h` |
| Move | Down | `j` |
| Move | Up | `k` |
| Move | Right | `l` |
| Move | Next word | `w` |
| Move | Previous word | `b` |
| Move | Start of line | `0` |
| Move | End of line | `$` |
| Move | Top of file | `gg` |
| Move | Bottom of file | `G` |
| Edit | Delete character | `x` |
| Edit | Delete word | `dw` |
| Edit | Delete line | `dd` |
| Edit | Copy (yank) line | `yy` |
| Edit | Paste | `p` |
| Edit | Replace character | `r` |
| Edit | Undo | `u` |
| Edit | Redo | `Ctrl + r` |
| Visual | Delete selection | `d` |
| Visual | Copy selection | `y` |
| Visual | Change selection | `c` |
| Visual | Indent / Unindent | `>` / `<` |
| Search | Search text | `/text` |
| Search | Next match | `n` |
| Search | Previous match | `N` |
| VS Code | Open file | `Ctrl + P` |
| VS Code | Go to line | `Ctrl + G` |
| VS Code | Command palette | `Ctrl + Shift + P` |
